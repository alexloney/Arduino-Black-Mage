# Arduino-Black-Mage
This is the source code for my Black Mage cosplay with LED eyes.

## Hardware
* [Ardiono Uno ($22)](https://store.arduino.cc/usa/arduino-uno-rev3)
* [2xAdafruit NeoPixel Shield for Arduino ($55.90)](https://www.adafruit.com/product/1430)
* [Membrane 1x4 Keypad ($2.95)](https://www.adafruit.com/product/1332)
* [Stainless Tin Conductive Thread ($6.95)](https://www.adafruit.com/product/640)
* [9V battery clip with 5.5mm/2.1mm plug ($2.95)](https://www.adafruit.com/product/80)

Total cost: $90.75
